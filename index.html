Welcome to the Lognormal Total Species Estimator Pages.

These pages have the PASCAL code and general explanations of the method for estimating the theoretical total species in a ecological assemblage

Compiler

The PASCAL code was originally compiled in Borland Turbo Pascal version 4, although the modern Open Pascal compiler will compile it as long as the Borland option is set.

Authors and Contributors

The original fortran code was created for the doubly truncated lognormal distribution by George Sugihara. The bootstrapping and additional Chi Square tests of the distribution were added by me, George D.F. (Buz) Wilson. If you want to work on this code, please contact me. I think it is a useful program that hasn't gotten much attention.
